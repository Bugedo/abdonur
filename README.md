## Super Admin (Panel General)

El super admin puede ver **todas las sucursales**, estadísticas globales y gestionar cualquier pedido.

| Rol          | Email                     | Contraseña      |
|--------------|---------------------------|-----------------|
| Super Admin  | superadmin@abdonur.com    | SuperAdmin123!  |

---

## Cómo ingresar al panel

1. Ir a **/admin**
2. Ingresar email y contraseña
3. Si sos admin de sucursal, vas a ver solo los pedidos de tu sucursal
4. Si sos super admin, vas a ver el panel general con todas las sucursales
